# Student Management System V2

My first Python project built while learning Python.

A simple Student Management System built using Python.

This project allows users to manage student records through a menu-driven interface. Student data is stored permanently using JSON file handling.

## Features

- Add Student
- View All Students
- Search Student by Roll Number
- Update Student Details
- Delete Student
- Count Total Students
- JSON Data Storage
- Exception Handling
- Duplicate Roll Number Validation

## Concepts Used

- Python Dictionaries
- Lists
- Loops
- Conditional Statements
- Functions
- Exception Handling
- File Handling
- JSON Module


## How to Run

1. Clone the repository

```bash
git clone https://github.com/singhgamer0050/First-Project-Student-Manajement-System-.git
```

2. Navigate to the project folder

```bash
cd First-Project-Student-Manajement-System-
```

3. Run the program

```bash
python main.py
```
